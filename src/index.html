        ASCII Converter is a simple utility for converting an image into
        richtext ASCII art. Simply choose the image to convert and press Submit.
        Read further for an explanation of the settings.
      

      

        ASCII Chars is a sorted list of ASCII characters to use in the image. It
        goes from lightest to darkest. Give this set a try: ░▒▓█
      

      

        Lightness is an adjustment to the overall lightness of the image. ASCII
        art can come out darker than the original image, so this slider can be
        used to adjust it.
      

      

        Size is the number of columns (rows if the input image is landscape)
        that the final output will contain.
      

      
Font Size is the font size of the ASCII characters.

    

    
    

      This utility is provided as-is.
